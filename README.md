# 🐾 G-Pet — Loja Virtual de Produtos Pet

**G-Pet** é uma aplicação web de e-commerce voltada para produtos pet, desenvolvida com foco em aprendizado e prática de novas tecnologias. O projeto oferece uma experiência completa de compra, desde a navegação por produtos até o checkout com integração ao Stripe.

 💡 Sobre o Projeto

O plataforma G-Pet é uma loja online de produtos para animais, permitindo ao usuário:

- 🐱 Navegar por categorias específicas (gatos, cães, peixes, aves);
- 🛒 Adicionar e remover produtos do carrinho;
- 💳 Finalizar o pagamento com Stripe;
- 🔐 Realizar cadastro e login com Firebase Authentication.

A ideia principal é proporcionar uma experiência de e-commerce moderna, com foco na usabilidade, funcionalidade real e integração de tecnologias modernas.

🚀 Desafios e Aprendizado

Este projeto foi uma jornada de aprendizado e superação de limites pessoais e técnicos. Entre os maiores desafios e conquistas, estão:

- 📌 Aprender e utilizar Node.js e Express.js para construir o backend;
- 📌 Sair da zona de conforto ao integrar o Stripe para pagamentos reais;
- 📌 Trabalhar com Firebase Authentication para criar um sistema de login seguro;
- 📌 Desenvolver e controlar um carrinho de compras funcional com `localStorage`;
- 📌 Lidar com sessões de pagamento, cancelamentos e confirmações de forma fluida;
- 📌 Criar um layout responsivo e amigável com HTML, CSS e JavaScript puros;
- 📌 Organizar o projeto em múltiplos arquivos e categorias reutilizáveis.

Foi um grande passo sair da teoria e colocar todo esse conhecimento em prática dentro de um projeto completo.
